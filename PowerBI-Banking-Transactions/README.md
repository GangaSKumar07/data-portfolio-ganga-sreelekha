**Power BI Banking Transactions Dashboard**
This dashboard analyzes customer banking transactions to uncover spending behavior, account patterns, and financial activity trends.
 
**Key Insights**
> 3Daily, weekly, monthly transaction volumes
> Top spending categories and merchant types
> Customer segmentation (age, income, location)
> High-value vs low-value transaction trends
